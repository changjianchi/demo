# demo

> 学习/练习js的一些例子，目前只有`es5`，后续后添加`es6`、`jQuery`核心解读、`nodejs`、测试用例、集成`CI`等例子

## 格式

要求在目标文件夹里创建当前功能的文件夹，比如：`es5/array-forEach/`，表示数组的`forEach`功能研究，该文件夹内要求有：

* `README.md`该功能的描述，如：参数说明、兼容性、小技巧、性能研究、常用场景等
* `index.js`功能写的例子
